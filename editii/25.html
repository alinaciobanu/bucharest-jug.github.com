  • Costin Chiriac - Apelează cod C/C++ din Java prin Java Native Interface (si prieteni)


    Java JNI (Java Native Interface) este o metod de a apela cod nativ (scris în C/C++ sau limbaj de asamblare) din Java. Atât codul nativ cât și codul compilat Java au avantaje și dezavantaje care le fac potrivite pentru diverse sarcini. Prezentarea își propune să prezinte soluția de apel de cod Java JNI cât si alte proiecte libere similare.

  • Costin Chiriac - Call C/C++ code from Java using Java Native Interface and friends

    Java JNI (Java Native Interface) is a framework allowing calls to and from native code (written in C, C++ or assembly language). Both bytecode and native code have advantages and drawbacks which make them interesting options for different tasks. We will also look at some open source projects that allow a simpler approach than JNI.
